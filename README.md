# Data Engineering Pipeline

A Rust-based data engineering project for CSV import, data transformation, and reporting.

## Project Structure

- **data/** - Sample CSV files
- **reports/** - Generated reports
- **src/** - Source code
  - `main.rs` - CLI entry point
  - `db.rs` - PostgreSQL connection
  - `importer.rs` - CSV import functionality
  - `transform.rs` - Data cleaning and transformation
  - `analytics.rs` - Data query operations
  - `report.rs` - Report generation
